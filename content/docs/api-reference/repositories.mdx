---
title: Repositories
description: Create, list, update, and delete Git repositories via the API.
---

## Create Repository

```http
POST /v1/repos
```

```json
{
  "slug": "my-repo",
  "description": "An example repository",
  "visibility": "private",
  "default_branch": "main"
}
```

**Response** `201`:

```json
{
  "id": "repo_abc123",
  "slug": "my-repo",
  "description": "An example repository",
  "visibility": "private",
  "default_branch": "main",
  "created_at": "2025-01-01T00:00:00Z"
}
```

## List Repositories

```http
GET /v1/repos?page=1&per_page=20
```

Returns an array of repository objects.

## Get Repository

```http
GET /v1/repos/:slug
```

Returns the repository object with an `is_empty` flag.

## Update Repository

```http
PATCH /v1/repos/:slug
```

```json
{
  "description": "Updated description",
  "visibility": "public",
  "default_branch": "develop"
}
```

## Delete Repository

```http
DELETE /v1/repos/:slug
```

Deletes the repository and all associated snapshots. Git objects in storage are cleaned up asynchronously.
